# ETF动量轮动策略 — 完整总结手册（含上涨质量过滤）

---

## 一、策略总览

### 1.1 策略名称
**双动量 + 上涨质量 + 低波 + 均线 混用策略**

### 1.2 策略定位
基于A股ETF市场的趋势跟踪轮动策略，通过"动量选股 + 质量过滤 + 低波加权 + 均线确认 + 积极/平衡双策略混用"，实现稳健增值。

### 1.3 核心逻辑
```
不买跌的（绝对动量）
  → 涨得稳的（上涨天数占比过滤"脉冲式上涨"）
    → 买最强的（相对动量排序）
      → 稳的多买（低波加权）
        → 趋势确认（均线过滤）
```

---

## 二、五大核心因子

### 2.1 双动量因子（选什么）

| 类型 | 作用 | 计算方式 |
|------|------|---------|
| 绝对动量 | 剔除弱势ETF | 动量 > 0 才纳入候选 |
| 相对动量 | 选出强势ETF | 按动量高低排序，选前N名 |

**动量计算公式**：
```
动量 = (当前净值 - N日前净值) / N日前净值 × 100%
```

### 2.2 上涨质量过滤（过滤"上窜下跳"）⭐新增

**核心思路**：不光看涨了多少，还要看"怎么涨的"——稳步上涨才是真的强。

**计算方式**：
```
上涨天数占比 = 动量窗口内上涨天数 / 总交易日数
```

**过滤规则**：
```
条件：上涨天数占比 > 50%
含义：动量窗口内，超过一半的交易日是上涨的
效果：排除靠少数几天暴涨堆出来的"脉冲式上涨"
```

**典型场景**：
```
ETF A（稳步上涨）：
  20日涨15%，12天涨8天跌 → 上涨占比60% → 通过 ✅

ETF B（脉冲式上涨）：
  20日涨18%，6天涨14天跌 → 上涨占比30% → 排除 ❌
  （靠少数几天暴涨堆出来，随时可能回落）
```

**在策略中的位置**：
```
绝对动量过滤 → 上涨质量过滤 → 相对动量排序 → 低波加权 → 均线过滤
```

### 2.3 低波因子（买多少）

**波动率计算**：过去N日日收益率的标准差

**权重分配**：
```
单只权重 = (1 / 波动率) / Σ(1 / 波动率)
```

**逻辑**：波动越小权重越高，降低组合整体风险。

### 2.4 均线因子（能不能买）

| 层级 | 作用 | 条件 |
|------|------|------|
| 个股均线 | 确认个股趋势 | 收盘价 > N日均线 |
| 大盘过滤 | 判断市场环境 | 中证A500收盘价 > M日均线 |

**大盘过滤效果**：
- 通过 → 允许最高100%仓位
- 未通过 → 最高50%仓位（或持有国债ETF）

### 2.5 混用策略（怎么分配）

| 策略类型 | 资金占比 | 定位 |
|---------|---------|------|
| 平衡型 | 70% | 稳健底仓，核心收益来源 |
| 积极型 | 30% | 进攻弹性，捕捉短线机会 |

## 四、中证A500 vs 沪深300

### 为什么选中证A500作为大盘过滤？

| 维度 | 沪深300 | 中证A500 |
|------|---------|----------|
| 成分股数 | 300只 | 500只 |
| 市值特征 | 超大型企业（大象级） | 中型企业（创新主力军） |
| 行业分布 | 金融+能源+消费为主 | 科技+医药+制造更均衡 |
| 与全市场相关性 | ~0.85 | ~0.92 |
| 策略匹配度 | 较低（主要覆盖大蓝筹） | 较高（与行业ETF更相关） |
| 信号灵敏度 | 较慢 | 更快 |

**结论**：中证A500更能代表A股整体，对市场情绪变化更敏感，与我们交易的ETF相关性更高。

## 五、每周执行流程

```
每周五收盘后执行

Step 1: 数据准备
  ├── 获取所有ETF最新净值
  ├── 计算动量、波动率、均线
  ├── 计算上涨天数占比
  └── 确认中证A500大盘状态

Step 2: 筛选（双动量 + 上涨质量 + 均线）
  ├── 第一层：绝对动量 > 0（只看涨的）
  ├── 第二层：上涨天数占比 > 50%（只看涨得稳的）
  ├── 第三层：收盘价 > 均线（只看涨势确认的）
  └── 第四层：相对动量排序（选最强的）

Step 3: 分别计算两个策略
  ├── 平衡型（10/20日动量 + 60日均线）
  │   └── 输出：持仓建议（3只以内）
  └── 积极型（5/10日动量 + 20日均线）
      └── 输出：持仓建议（2只以内）

Step 4: 大盘仓位调整
  ├── 中证A500 > 120日均线 → 目标仓位100%
  │   ├── 平衡型: 70%
  │   └── 积极型: 30%
  └── 中证A500 < 120日均线 → 目标仓位50%
      ├── 平衡型: 50%（或国债ETF）
      └── 积极型: 0%（清仓或国债）

Step 5: 合并结果 + 低波加权
  ├── 低波因子分配权重（波动小的多买）
  ├── 同一ETF → 取较高权重
  ├── 不同ETF → 按比例分配
  └── 检查单只权重是否超限

Step 6: 冷静期检查
  ├── 新买入ETF未满3个交易日 → 不换仓（止损例外）
  └── 持仓ETF从高点回撤>8% → 无条件清仓

Step 7: 周一开盘执行调仓
```

## 六、风控机制

### 6.1 止损规则

| 条件 | 操作 |
|------|------|
| 单只ETF从高点回撤 > 8% | 无条件清仓 |
| 整体组合回撤 > 15% | 降至50%仓位 |

### 6.2 仓位限制

```
单只ETF上限：
  ├── 平衡型: 40%
  └── 积极型: 50%

整体仓位：
  ├── 大盘强势: 最高100%
  └── 大盘弱势: 最高50%
```

### 6.3 冷静期

新买入ETF至少持有3个交易日（止损例外），减少无效交易。
